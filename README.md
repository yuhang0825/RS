# RS
* 此项目基于开源项目以及一些博客学习
* 兰州大学大二物理系，第一次尝试开发软件，做了一个简单的兰州大学教务系统移动端，目前已经完成课程表以及不确定度的计算。
* 网页请求采用okhttp
* 网页解析采用Jsoup

## 课程表
* 借用[此开源项目](https://github.com/chen2174471/TimeTableLayout)的课程表页面布局
* 支持添加两个人的课程信息
* 导入准确度目前没有什么问题，不支持编辑

## 不确定度的计算
* 此方式可能仅适合兰州大学物理实验！
* 只能计算直接测量量的不确定度。
* 系数PN采用0.683对应的数值，内置实验组数应在3—10组。
* UB的计算方式为（最小刻度值）/（根号3）
* 请将实验测量若干组数字，输入时，以加号（+）隔开
